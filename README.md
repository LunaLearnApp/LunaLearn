# LunaLearn

LunaLearn is an open-source project primarily developed in ReactJS, designed to provide an interactive and user-friendly learning experience.

## Features

- Built with ReacrJS for reliability and scalability
- HTML-based UI for a smooth user experience
- Modular and extensible architecture

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Luna023/LunaLearn.git
   cd LunaLearn
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the project:**
   ```bash
   npm start
   ```
